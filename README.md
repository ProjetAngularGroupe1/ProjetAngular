# AngularProject

## Launch Backend

```cmd
cd backend/
```

```cmd
npm install
```

```cmd
npm run start
```

[localhost:3000]()

## Launch Frontend

```cmd
cd frontend/
```

```cmd
npm install
```

```cmd
npm run start
```

[localhost:4200]()
