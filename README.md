# AngularProject

## Launch Backend

```cmd
cd backend/
```

```cmd
npm install
```

```cmd
npm run start
```

```cmd
localhost:3000
```

## Launch Frontend

```cmd
cd frontend/
```

```cmd
npm install
```

```cmd
npm run start
```

```cmd
localhost:4200
```
